# Job Hub Center
I coded "jobhubcenter.com" from scratch, and the purpose of this site is to make it easier for people to find jobs.

## Live
Live : https://jobhubcenter.com/en

## Built with
- Next.js 14 | Server Actions
- TypeScript
- MongoDB | Mongoose
- Reacti18

## Features
- Advanced Search and Filtering Options:
Filtering by Category: Ability to filter job listings by category.
Search Engine: Search jobs by job title or other keywords.
- Multi-language Support:
Language Options: Ability for users to switch between Azerbaijani and English languages.
- Personal Job Listing Creation: Users can create their own job listings
- User-friendly Interface and Experience: Modern and user-friendly UI/UX design. Responsive design ensuring compatibility across devices.
